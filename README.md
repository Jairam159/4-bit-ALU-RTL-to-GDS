Design and implementation of a 4-bit ALU from RTL to GDSII using OpenLane, OpenROAD, Sky130 PDK and KLayout.

# 4-Bit ALU Design: RTL to GDSII using OpenLane

## Project Overview

This project demonstrates the complete ASIC physical design flow of a **4-bit Arithmetic Logic Unit (ALU)** using the **OpenLane** open-source digital ASIC flow and the **SkyWater SKY130 Process Design Kit (PDK)**.

The design starts from **RTL coding in Verilog**, followed by **functional verification**, **logic synthesis**, **floorplanning**, **placement**, **clock tree synthesis (CTS)**, **routing**, and finally **GDSII generation** for fabrication-ready layout visualization.

---

## Objectives

- Design a 4-bit ALU using Verilog HDL.
- Verify the functionality using a Verilog testbench.
- Generate simulation waveforms using GTKWave.
- Perform logic synthesis using Yosys.
- Implement the physical design flow using OpenLane/OpenROAD.
- Generate the final GDSII layout using the SKY130 PDK.
- Visualize the final layout in KLayout.

## ALU Specifications

- Data Width: 4-bit
- Inputs:
  - A[3:0]
  - B[3:0]
  - Carry Input
- Outputs:
  - Result[3:0]
  - Zero Flag

---

## Supported Operations

| Opcode | Operation |
|---------|-----------|
|000|Addition|
|001|Subtraction|
|010|Bitwise AND|
|011|Bitwise OR|
|100|Bitwise XOR|
|101|Bitwise NOT|
|110|Left Shift|
|111|Right Shift|
